# Setup


## Front
Navigate to: `/charlotte-research-connect`

Execute `npm i` in cmd/bash

Execute `npm start` in cmd/bash to start development server

## Back
Navigate to `/Backend`

Execute `npm i` in cmd/bash

Execute `nodemon .` in cmd/bash to start development server
